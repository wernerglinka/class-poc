# Cheatsheet to review a class submission

## 1 Open the Google Sheet at https://docs.google.com/spreadsheets/d/1qF4-CWEIqupjdIblx_1p7URP8OZbtWbOKrN9ffKr3yk/edit?gid=953093648#gid=953093648

## 2 Open the Offerings tab

## 3 Click any cell in the row you want to review

## 4 Select Review >> Open edit modal in the sheet menubar
A large editing window opens with every detail of the class in labeled fields. Long text boxes grow to fit their content. The class dates appear at the bottom under Sessions, where you can correct, add, or remove a session. Click Submit to save, or Cancel to close without changing anything.

## 5 Approve the class
Once all edits are done and the class page is ready to be published, type **yes** into the approved column (in the sheet or in the edit modal). Then select Review >> Publish site; the class appears on the CPC website within a couple of minutes.
