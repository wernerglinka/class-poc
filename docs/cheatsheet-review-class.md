# Cheatsheet to review a class submission

## 1 Open the Google sheer at https://docs.google.com/spreadsheets/d/1qF4-CWEIqupjdIblx_1p7URP8OZbtWbOKrN9ffKr3yk/edit?gid=953093648#gid=953093648

## 2 Open the Offerings tab

## 3 Select Review >> Open row viewer in the sheet menubar

## 4 Click on the row to review
You will now see all info in a right sidebar. Click on any text and it will change into an editable field. M ake your edit and a Save button plus a Discard link will appear at the bottom of the sidebar.

## 5 Approve the class
Once all edits are done and the class page is ready to be published, type **yes** into the approved column. The page will be added to the CPC website during the next build.
